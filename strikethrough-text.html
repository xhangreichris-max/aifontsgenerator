What Is a Strikethrough Text Generator?

Strikethrough text uses Unicode combining characters to draw a horizontal line through standard letters, making the text appear crossed out. The strikethrough text generator adds the Unicode combining long stroke overlay to each character in your input. The result is readable text with a visible line through it, which reads visually as deleted, sarcastic, or ironic depending on the context. It is popular on Discord for sarcasm in chat messages, on Instagram for before-and-after style bio formatting, and on TikTok for display names and captions with a self-aware tone. The output is plain Unicode that pastes anywhere without formatting support from the platform.

How to Use This Strikethrough Text Generator

Type your text into the input field and the strikethrough text generator applies the crossing overlay to every character instantly. Copy the output and paste it into Discord, Instagram, TikTok, or any other platform. The strikethrough line appears on every device that reads the text. This is different from Discord's own markdown strikethrough, which only works in chat messages. Unicode strikethrough works in name fields, bios, and anywhere else that accepts text input.

Strikethrough Text Generator for Instagram

Strikethrough text in an Instagram bio adds visual variation and can signal irony or a before-and-after format. A struck-out word followed by its replacement reads as a personality note: something you used to be versus what you are now. The formatting also works as a visual separator. Instagram does not support markdown, so Unicode strikethrough is the only way to get struck-out text in bio or name fields.

Strikethrough Text Generator for Discord

Discord has its own markdown strikethrough using double tildes, but that only works in chat messages. In nickname fields, bios, and status messages where markdown does not apply, Unicode strikethrough is the only option for struck-through text. A strikethrough nickname in Discord reads as sarcastic or self-deprecating, which works well in casual or humorous servers. The effect also works in Discord status messages for ironic or running-joke statuses.

Platform Compatibility Guide

Discord, Instagram, TikTok, Twitter, and WhatsApp all support Unicode strikethrough text. It displays as struck-through text on every device using these platforms. Facebook renders Unicode strikethrough correctly in posts and comments. Most gaming platforms have inconsistent support for the combining overlay Unicode character. PUBG Mobile and Free Fire may or may not display it correctly depending on the client version. For social media use, strikethrough is one of the most reliably compatible Unicode styles available.

Why Use Unicode Strikethrough Text Generator?

Unicode strikethrough uses a combining character called the combining long stroke overlay. Combining characters attach to the character before them and add a visual mark. The long stroke overlay was designed for linguistic and mathematical notation. When applied to every character in a word, it produces the appearance of crossed-out text. Because it is a combining Unicode character, it works without any formatting support from the app.
